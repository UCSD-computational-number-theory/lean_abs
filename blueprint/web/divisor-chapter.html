1 The generalized Dirichlet divisor problem

Definition 0.0.1 Divisor sum exponents
#

Let \(k \geq 1\) be a fixed integer. Then, \(\alpha _k\) is the best (fixed) exponent for which one has the asymptotic

\[ \sum _{n \leq x} d_k(n) = x P_{k-1}(\log x) + O(x^{\alpha _k+o(1)}) \]

for unbounded \(x {\gt} 0\), where \(P\) is an explicit polynomial of degree \(k-1\) and \(d_k(n) := \sum _{n_1 \dots n_k=n} 1\) is the \(k^{\mathrm{th}}\) divisor function. The implied constant may depend on \(k\).

Of course, one can also give a non-asymptotic definition: \(\alpha _k\) is the least exponent such that for all \(\eps {\gt} 0\) there exists \(C {\gt} 0\) such that

\[ |\sum _{n \leq x} d_k(n) - x P_{k-1}(\log x)| \leq C x^{\alpha _k + \eps } \]

for all \(x \geq C\).

In the case \(k = 1\), the problem is trivial. In particular:

Lemma 0.0.2 \(d_1\) exponent
#

Uses

  • Definition 0.0.1

One has \(\alpha _1=0\).

Proof ▼

Follows from \(\sum _{n \le x}1 = x + O(1)\).

On the other hand, the value of \(\alpha _k\) is an open problem for all \(k \ge 2\). Unconditionally, the following lower-bound on \(\alpha _k\) is known to hold.

Lemma 0.0.3 Lower bound on \(\alpha _k\)
#

Uses

  • Definition 0.0.1

For all \(k \geq 1\), one has

\[ \alpha _k \geq \frac{1}{2} - \frac{1}{2k}. \]
Proof ▼

See Hardy [ .

Multiple authors [give cite] have refined Hardy’s result but only in the \(o(1)\) term in the exponent. It is conjectured that this lower bound on \(\alpha _k\) is in fact an equality [ . Amongst other consequences, this conjecture implies the Lindelöf hypothesis [ .

Conjecture 0.0.4 Generalised Dirichlet divisor problem conjecture
#

For all \(k \geq 1\), one has

\[ \alpha _k = \frac{1}{2} - \frac{1}{2k}. \]
